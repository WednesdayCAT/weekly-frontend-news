## 🚀 快速使用

### 订阅周刊

1. 给仓库点个 ⭐ Star，第一时间收到更新通知
2. 每周一 8:00 来仓库查看最新周刊
3. 欢迎提交 Issue/PR 补充优质资讯

### 本地运行

```bash
# 克隆项目
git clone https://github.com/WednesdayCAT/weekly-frontend-news.git

# 安装依赖
npm install

# 生成周刊
npm run publish
```
