# 📚 weekly-frontend-news
每周一自动更新的前端技术周刊，帮你高效掌握前端趋势。

## ✨ 项目亮点
- 🤖 自动化生成：GitHub Actions 定时爬取官方博客/技术社区，每周一 8:00 自动发布；
- 📚 结构化整理：框架更新/生态工具/开源项目/行业实践四大模块，清晰易懂；
- 🔒 内容精选：过滤低价值信息，只保留对开发者有实际帮助的资讯；
- 📦 开源共享：所有内容基于 MIT 协议开源，可自由复用与二次分发。

## 📅 最近更新
- [{week_num} 前端周刊](链接)：重点更新 Vue 3.5 性能优化、React 19 发布预告

## 📌 如何使用
1.  给仓库点个 ⭐ Star，第一时间收到更新通知；
2.  每周一 8:00 来仓库查看最新周刊；
3.  欢迎提交 Issue/PR 补充优质资讯，一起共建社区。
